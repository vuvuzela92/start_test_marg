# Документация скриптов

---

### Описание скриптов в main

**add_new_items.py**
Добавление товаров в Гугл таблицы.

**adv_spend.py**
Обновление таблицы adv_spend_new (сбор и актуализация рекламных расходов).

**autopilot_daily.py**
Обновление данных в Панели Управления (ПУ) за предыдущие дни.

**autopilot_hourly.py**
Обновление данных в Панели Управления (ПУ) за текущий день (каждые 30 минут).
Если данные выгружаются неправильно, проверьте ПУ на наличие дубликатов.

**avg_position_to_db.py**
Добавление данных по средней позиции товаров за предыдущий день в базу данных.

**balance_history.py**
Добавление остатков ФБО (FBO) в таблицу базы данных balance_history.

**china_buy.py**
Обновление таблицы «Расчет поставки Китай» (отдел Закупок).
Ломался только если руками что-то меняли.

**daily_penalties_to_gs.py**
Обновление листа «Штрафы» в Панели Управления и файле Условного расчета.

**db_data_to_purch_gs.py**
Обновление листов «Заказы_поставщиков_1С» и «Приходы_1С» в Google Таблицах.

**deductions_to_db.py**
Выгрузка данных по удержаниям из WB API в базу данных.

**delete_items.py**
Удаление товаров из Гугл таблиц.

**expenses_gs_to_db.py**
Парсинг расходов из отчета бухгалтера и их добавление в БД (таблица expenses).

**feedbacks_to_db.py**
Парсинг и сохранение всех отзывов WB в базу данных (таблица wb_feedbacks).

**feedbacks_to_gs.py**
Обновление информации по отзывам в Google Таблицу «Расчет закупки Россия».

**make_wb_pay_daily.py**
Добавление новой записи на лист «ВБ_к_оплате» в файле Условного расчета.

**market_3.py**
Скрипт по Висячим 1.

**market_status_from_db.py**
Скрипт по Висячим 2.

**migration_data_to_hang.py**
Скрипт по Висячим 3.

**net_profit_from_orders.py**
Расчет чистой прибыли по заказам и выгрузка данных в базу данных.

**new_adv.py**
Скрипт по Воронке продаж WB.

**purchase_price_update.py**
Обновление закупочных цен в Сопосте.

**rate_of_return.py**
Обновление таблицы по рентабельности.

**remains_report_update.py**
Выгрузка данных о стоимости текущих остатков в Google Таблицу «Стоимость остатков».

**wb_chats.py**
Выгрузка чатов WB в БД.

**wb_missing_supplies_goods_to_db.py**
Добавление недостающих данных по товарам в поставках в базу данных (не регулярный).

**wb_stocks.py**
Добавление актуальных остатков склада ФБС (FBS) в базу данных.

**wb_supplies_to_db.py**
Обновление общей информации по поставкам в базе данных.
